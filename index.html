Fallen

Fallen is a top-down dungeon crawler game made in unity. The game will have similar game mechanic aspects to The Binding Of Isaac. The game is currently in the prototype phase. You can view my progress on the game on my GitHub.

Made with Unity

Fallen is being made with unity, as such when the game gets its first public beta's and demo's it can be used on both Windows and Mac.

Settings Menu

Fallen is going to have an in-depth settings menu to support as many devices as possible.

Supported Platforms

Fallen will release with a Windows and Mac verison.

In-Game Currency

The game will have a currency where you can purchase costumes and colors for the characters. In the dungeon you can find currency where you can choose to not use it and recieve it once the game ends or use it to help you in the game.
